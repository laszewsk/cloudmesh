cloudmesh.util.cm_table module

Prints tables in html, text, or with right alighned columns

class cloudmesh.util.cm_table.cm_table[source]

Class to print a table in various formats.

format = HTML | ASCII | “%20s”

type is not used and wil be in future removed

columns = number of columns

structure is an array of rows

header = True the firts row is a headline

create(structure, columns, format=None, type=None, header=True)[source]

creates a table from a list of rows, where the row is also a list. The format can be specified and may be HTML | ASCII | formatstring. A format string may for example be ‘%20s’ which prints the string in a field 20 spaces wide.
